## Installation

TYHWaterMark is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'TYHWaterMark',:git=>'https://github.com/pencilCool/TYHWaterMark.git'
```
